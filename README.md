# Proyecto Final: Análisis de Ventas y Segmentación RFM
**Autor:** Nicolás Vera  
**Curso:** Herramientas Básicas para el Análisis de Datos  
**Cohorte:** 2026

## Descripción
Este proyecto analiza el comportamiento de ventas y la segmentación de clientes mediante el modelo RFM (Recencia, Frecuencia y Valor Monetario). Se utilizó Python para el procesamiento de datos y Power BI para la creación de un dashboard interactivo.

## Estructura del Repositorio
- **/data**: Datasets originales y procesados (CSV).
- **/notebooks**: Código en Python (.ipynb).
- **/dashboard**: Archivo de Power BI (.pbix) y reporte en PDF.
- **/figs**: Capturas de pantalla de las visualizaciones.

## Conclusión
Se identificaron con éxito los grupos de clientes VIP y Leales, permitiendo una visión clara de la rentabilidad por categoría de productos.
